About TutoCito CMS

TutoCito CMS

TutoCito CMS is a React-based Single Page Application (SPA) with a .NET Core backend, REST API, WYSIWYG editor, and full SEO support.

  • ASP.NET Core 8 — backend framework
  • React + Recoil — frontend library
  • Bootstrap 5 — responsive design

Ideal for blogs, business websites, and e-commerce. Open source (MIT), supports VPS, and extensible with custom modules.

About the Project

TutoCito CMS

The goal is to create a powerful, accessible, and universal tool for publishing online — suitable for both beginners and professionals.

Built on ASP.NET Core 8 and React, TutoCito CMS is actively maintained and regularly updated. It supports REST API, role-based access, SEO, and mobile responsiveness.

Whether you're a developer or a business owner, you can quickly launch and manage a modern website with full control over content and design.
